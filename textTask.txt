Frontend Developer API Documentation
Base API URL: https://100m.uz
1. Registration
Endpoint: POST /register
Expected Payload:
•	- username
•	- fullname
•	- email
•	- password
•	- template_name
2. Login
Endpoint: POST /login
Expected Payload:
•	- username
•	- password
•	- remember_token
